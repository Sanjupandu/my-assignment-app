# ShopHub User Stories

1. As a new user, I want to create an account so that I can save my favorite products.
2. As a registered user, I want to log in with my email and password so that I can securely access my profile.
3. As a user, I want to see an error message if I enter the wrong login credentials so that I know to try again.
4. As a shopper, I want to view a home screen with a list of products so that I can browse the store.
5. As a shopper, I want to tap on a specific product so that I can see more detailed information about it on a separate screen.
6. As a user, I want the app to fetch real-time product data from an external API so that the catalog is always up-to-date.
7. As a user, I want my preferences to be saved in local storage so that the app remembers them the next time I open it.
8. As a user, I want to access a settings menu so that I can configure my app preferences.
9. As a user, I want to receive push notifications so that I am alerted to new sales even when the app is closed.
